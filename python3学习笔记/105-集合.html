集合¶

是一个无序 不重复可变元素集(但元素必须是不可变的元素)，主要用作去重和关系测试，和字典一样使用{}，所以创建空集合不用{}，用set()

基本操作¶

In [2]:
x = set('aabb')
x
Out[2]:
{'a', 'b'}

如果创建集合时有可变元素则会报错

x = {1,[1, 2]}
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: unhashable type: 'list'

unhashable --> 不可哈希 --> 可变
hashable --> 可哈希 --> 不可变

关系测试¶

In [12]:
x = {1, 2, 3}
y = {2, 3, 4}

x | y  # --> 并集
x & y  # --> 交集
x - y  # --> 差集
x ^ y  # --> 对称差集

{1, 2, 3, 4}
Out[12]:
{2, 3}
Out[12]:
{1}
Out[12]:

add, update¶

In [20]:
x = {1, 2, 3}
x.add('abc')
x  # -->

x.update('abc')  # 拆分更新
x  # -->
Out[20]:
{'abc', 1, 2, 3}
Out[20]:
{'abc', 1, 2, 3, 'c', 'a', 'b'}

set.pop()¶

随机删除元素并返回该元素，不存在则报错

In [15]:
x = {1, 2, 3}
x.pop()  # -->
Out[15]:
1

set.remove(obj)¶

删除指定元素，若集合中没有该元素则报错，修改原集合没有返回值

In [14]:
x = {1, 2, 3}
x.remove(1)

x
Out[14]:
{2, 3}

set.discard(obj)¶

删除指定元素，集合中没有该元素不报错，修改原集合没有返回值

In [21]:
x = {1, 2, 3}
x.discard(1)
x             # -->
x.discard(4)  # do nothing
Out[21]:
{2, 3}
