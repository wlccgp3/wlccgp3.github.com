## 列表相减

解法一:用for循环
```
x = [1, 2, 3, 3, 2, 1]
y = [1, 4]
z = []

for i in x:
    if i on in y:
        z.append(i)

print(z)            #--> [2, 3, 3, 2]
```

解法二:用列表解析
```
>>> x = [1, 2, 3, 3, 2, 1]
>>> y = [1, 4]
>>> z = [i for i in x if i not in y]
>>> z                                #--> [2, 3, 3, 2]
```
