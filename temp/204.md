
## 递归
> 递归（recursion），又译为递回，在数学与计算机科学中，是指在函数的定义中使用函数自身的方法。

根据斐波那契数列的定义，可以直接写成这样的斐波那契数列递归函数。
```
#!/usr/bin/env python3
# coding=utf-8

def fib(n):
    """
    This is Fibonacci by Recursion.
    """
    if n==0:
        return 0
    elif n==1:
        return 1
    else:
        return fib(n-1) + fib(n-2)

n = int(input('输入正整数:'))
print(fib(n))
```

解析当n = 4时：
```
fib(4) = fib(3)       +     fib(2)
             ↓                  ↓
      fib(2) + fib(1)    fib(1) + fib(0)
          ↓
   fib(1) + fib(0)

```

相当于fib(4) = 3*fib(1) + 2*fib(0) -> 3

> 上面的代码每次递回下一级函数，都要判断一次判断一次if和elif

相当于

优化代码
```
#!/usr/bin/env python
# coding=utf-8

d = {0:0, 1:1}    

def fib(n):
    if not n in d:    
        d[n] = fib(n-1) + fib(n-2)
    return d[n]

n = int(input('输入正整数:'))
print(d(n))                 
```

> 用递归函数要小心，因为很容易陷入死循环
